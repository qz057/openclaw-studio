# Security Policy

## Supported Versions

The current public line is the `v0.1.0-preview` unsigned preview.

## Reporting A Vulnerability

Use GitHub Security Advisories if enabled for the repository. If advisories are not enabled yet, contact the maintainer privately before publishing exploit details.

## Signing Status

The preview release artifacts are unsigned. The trusted signing track is prepared separately and remains blocked until trusted code-signing certificate material is available.

Do not commit certificate files, private keys, token credentials, or signing passwords.
